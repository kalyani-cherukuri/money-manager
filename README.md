# 💰 Money Manager App

A full-stack Money Manager application built using Spring Boot (Backend) and React (Frontend).  
This app helps users manage their income and expenses efficiently.

---

## 🚀 Features

- User Authentication (Login / Signup)
- Add Income & Expenses
- Dashboard with financial overview
- Filter transactions by date
- Search expenses
- Track spending habits

---

## 🛠️ Tech Stack

Frontend:
- React.js
- Axios

Backend:
- Spring Boot
- Spring Security (JWT)
- MySQL

---

## 📁 Project Structure

money-manager/
 ├── frontend/
 └── backend/

---

## ⚙️ Setup Instructions

### Clone the repo
git clone https://github.com/kalyani-cherukuri/money-manager.git
cd money-manager

### Run Backend
cd backend
mvn spring-boot:run

### Run Frontend
cd frontend
npm install
npm start
